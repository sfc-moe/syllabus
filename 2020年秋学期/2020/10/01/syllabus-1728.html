科目概要

本講義では、データセキュリティを、概念的、技術的、運用的、政策的な面から概観する。概念編では、データのもつ価値と、セキュリティの目標である機密性、完全性、可用性の考え方を学ぶ。技術編では、暗号理論や鍵管理を中心に、ベースとなる技術用語とその理解を深める。運用編では、データセキュリティの社会実装例である、公共システムや金融システムなどでの実例を深める。政策編では、日本、米国、欧州のデータセキュリティに関わる最新状況を概観する。

In this lecture, data security will be explained from a conceptual, technical, operational and policy perspective. The concept section provides the value of data and the concepts of security goals such as confidentiality, integrity, and availability. The technical section focus on cryptographic theory and key management and deepen the understanding of the basic technical terms and their understanding. The operation section provides actual examples in public systems, financial systems and control systems, which are examples of social implementation of data security. The policy section provides an overview of the latest status regarding data security in Japan, the United States, and Europe.

授業計画

第1回 データセキュリティの歴史
[Data security history]

データセキュリティの萌芽は、ギリシャ・ローマ、中国・孫氏、近代戦争と暗号、東西冷戦などにみられる。現代セキュリティにつながる歴史を振り返る。

The emergence of data security can be seen in Greece, Rome, China, Sun, modern war and crypto, and the Cold War. Looking back on the history of modern security.

手塚　悟:Satoru TEZUKA

第2回 セキュリティ目標、可用性・完全性・機密性
[What are security goals, availability/integrity/confidentiality]

データを必要なときにデータを使えることという可用性の目標、および、データが改変されていないこと、削除されていないことという完全性の目標、および、データを権限のない者に見せないことという機密性の目標を、身の回りの実例を交えて説明する。

The goal of availability, that data is available when it's needed, the integrity of data that hasn't been tampered with, and that it hasn't been deleted, and the confidentiality of not showing it to unauthorized people. Provide familiar goals with real-life examples.

手塚　悟:Satoru TEZUKA

第3回 共通鍵暗号、公開鍵暗号
[Common key cryptography, public key cryptography]

暗号理論の基礎的な素養となる、共通鍵暗号、公開鍵暗号の技術を説明する。

The techniques of common key cryptography and public key cryptography, which are the basic knowledge of cryptography, are explained.

手塚　悟:Satoru TEZUKA

第4回 一方向性関数、デジタル署名、タイムスタンプ
[One-way function, digital signature, time stamp]

暗号理論の応用的な素養となる、一方向性関数、デジタル署名、タイムスタンプなどの技術を説明する。

We will explain technologies such as one-way functions, digital signatures, and time stamps, which are applied foundations of cryptography.

手塚　悟:Satoru TEZUKA

第5回 暗号解析と暗号強度、鍵管理、暗号通信プロトコル
[Cryptography analysis and encryption strength, key management, cryptographic communication protocol]

暗号理論をシステム的に成立させるための、暗号解析、暗号強度、鍵管理などの技術を説明する。暗号理論の実装例としての、TLS(Transport Layer Security)やVPN(Virtual Private Network)などの暗号通信プロトコルを説明する。

Techniques such as cryptographic analysis, cryptographic strength, and key management for systematically establishing cryptographic theory will be described. Cryptographic communication protocols such as TLS (Transport Layer Security) and VPN (Virtual Private Network) will be explained as implementation examples of cryptography.

手塚　悟:Satoru TEZUKA

第6回 公共システム
[Public system]

データセキュリティを前提として社会的に使われている、公共システムの実例を説明する。公的個人認証サービス、マイナンバーなど。

Explain the actual examples of public systems that are used socially on the premise of data security. Public personal authentication service, my number, etc.

手塚　悟:Satoru TEZUKA

第7回 金融システム
[Financial system]

データセキュリティを前提として社会的に使われている、金融システムの実例を説明する。ATM、クレジットカード決済システムなど。

Explain an example of a financial system that is used socially assuming data security. ATM, credit card payment system, etc.

近藤　賢郎:Takao KONDO

第8回 制御システム
[Control system]

データセキュリティを前提として社会的に使われている、制御システムの実例を説明する。スマートグリッド、自動運転車など。

An example of a control system used socially on the premise of data security is explained. Smart grids, self-driving cars, etc.

近藤　賢郎:Takao KONDO
