科目概要

本講義ではＷｅｂのメディアとしての特性を理解した上で、企業等の組織におけるＷｅｂ構築管理の一連のプロセスについて学びます。企業等の組織におけるＷｅｂ構築プロジェクトを疑似体験し実践的なＷｅｂ構築管理に必要な知識・スキルを修得します。また同時に各種ＷｅｂアプリケションやＣＳＭ、などのWebで活用されるテクノロジーの基礎的事項について理解を深めます。

本校講義では企業等組織におけるＷｅｂ制作の現場において求められる知識及びスキルの体系をカバーするよう努めています。想定される履修者は企業や組織におけるＷｅｂ製作・運用担当者、Ｗｅｂテクノロジの開発者や研究者、マーケティング担当者、コンサルタントなどのキャリアを目指す学生です。

2020年度はWebメディア企業であるサイバーエージェント社の協力を得て現実のWebメディアやサービスの制作プロセスや分析等、実際のサービス開発に即したプログラムを取り入れる予定です。

Through this course students will learn processes of web design and development in corporate environment with understanding of nature of web media.

This course covers design principle, related technologies, tools, standard process of web design and its management.
Students experience web development projects as a group work with professional team setting.

This course is intended to be registered by students who seek carrier in web developer, manager, designer, researcher, marketing, and consultant etc.

提出課題・試験・成績評価の方法など

個人でのWebサイトの制作及びグループでのWebサイトの制作に関する実習を行います。
成績評価は授業への出席、各制作物及びその過程に関するレポート及びグループワークにおける相互評価により決定します。

Individual web project and group web project would be conducted.
Grades for this class would be determined by attendance to the class, out come of web project, reports on the projects and mutual evaluation in the group work.

履修上の注意

本講義では参加学生のグループワークへの貢献を重視しており履修者は原則として全ての授業に出席することを前提としています。

本授業はオンラインで実施します。
グループワークを多くコミュニケーションを円滑なものとするため履修者は原則としてWebカメラをONにした状態で履修していただきます。また音声トラブル防止のためイヤホンマイクまたはWeb会議用スピーカーの使用を必須とします。（パソコン本体内蔵のスピーカーおよびマイクの使用は原則禁止とします。）

This course require attendance of all class hours since contribution to the group project has significant impact to the success of the course.

授業計画

第1回 講義概要・Web制作管理の基礎（Web制作プロジェクト説明）
[Course Introduction, fundamental of web design]

本講義の概要、グループワークとして行うWeb製作の説明、Web製作管理の一般的な内容を解説します。

Introduction of the course.
Explain fundamental of web design

第2回 環境構築、Webデザインのプロセス、チームビルディング
[System setup, web design process, team building]

個人のWeb製作のための環境構築について解説します。
またグループ制作をチームとそして進めていくための体制づくりを行います。

System setup for individual web development.
Explain web design process.
Team building for the group work.

第3回 Web制作実習（個人・グループ）
[Web design project (as individual and group )]

Web制作実習に向けたグループの顔合わせ
個人製作作業を進めます。

Team building for group project.
Preparation for individual project.

第4回 Web制作実習（個人）
[Web design project (individual)]

個人でのWeb製作の実習作業を進めます。

Conduct individual web design project.

第5回 企画発表（企画・設計・プロジェクトマネジメント）
[Proposal presentation (Project plan, design and project management)]

グループでのWeb製作に関する企画発表のプレゼンテーションを行います。

Conduct proposal presentation for group project.

第6回 Webの要素技術（サーバ、ブラウザ）
[Web related technologies (Server, browser etc.)]

Webの要素技術に関する解説を行います。

Explain web related technologies.

第7回 Web制作実習（グループ）2
[Web design project (Group) 2]

グループでのWeb制作活動を行います。

Conduct web design project activities as group.

第8回 Web運用・マーケティングSEOSEM
[Web operation, Marketing SEO/SEM]

グループ制作のリリースに向けWebの運用マーケティングSEO/SEM等について解説します。

Toward release of web project explain operation of web site, marketing, and SEO/SEM.

第9回 中間発表
[Interim Presentation]

グループ制作課題に関する中間発表を行います。

Interim Presentation for group web project

第10回 Web制作実習（グループ）３
[Web design project (group) 3]

グループでのWeb制作活動を行います。

Conduct group work activity for the web design project.

第11回 Web制作実習（グループ）4
[Web design project (group) 4]

グループでのWeb制作活動を行います。

Conduct group work activity for the web design project.

第12回 Web制作プロジェクト最終発表
[Final Presentation for the web development project.]

グループ制作課題に関する最終発表を行います。

Conduct final presentation of web design project.

第13回 グループワーク実習１
[Group work activity 1]

授業時間外でのグループワーク活動としてWeb製作の実習を行います。

Group work activity for the project out side of the regular class.

第14回 グループワーク実習２
[Group work activity 2]

授業時間外でのグループワーク活動としてWeb製作の実習を行います。

Group work activity for the project out side of the regular class.

第15回 グループワーク実習３
[Group work activity 3]

授業時間外でのグループワーク活動としてWeb製作の実習を行います。

Group work activity for the project out side of the regular class.
