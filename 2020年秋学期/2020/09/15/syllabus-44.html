研究会概要

目的・内容

ソフトウェアを使って様々な問題を解決しましょう。大学や身近なところにも、解決した問題はいっぱいあります。そのような問題をソフトウェアを作ることで解決しましょう。

解決する問題はグループ毎に提案ベースで行います。2020年春学期は、次のようなテーマのグループができています。

- YouTubeのクローンの作成
- 就職活動におけるつながりを作る携帯アプリ
- プログラミング授業の演習状況の可視化
- グループでの終電の調整アプリ
- EDINETデータの分析および可視化
- LINEリマインダ
- 講義案内システム
- レストランレビューシステム

There are various problems around us which we can solve by creating a software system. Each group proposes a problem to solve and creates a system for it. In 2020 spring semester, the following systems are currently tackled by the groups.

- YouTube close
- Application for connect students seeking jobs
- Visualization of programming exercises
- Last train adjustment in a group
- EDINET data analysis and visualization
- LINE reminder
- Lecture recommend system
- Restaurant review system

評価方法

提案システムの完成度によって評価します。

Evaluated by the system created by the group.
